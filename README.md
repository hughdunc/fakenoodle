# Fake Noodle (Impasta)

Fake Noodle is an imposter game, but every other app is paid for new words and features so I made my own

** If you have any ideas or feedback, please please say so! Best place to say it is Github Issues**

## Where to get Wordsets?

Go to the `wordsets` git branch and read the README, it will show how to get, create, or post wordsets

## What is the "Basic 1000" wordset?

This base wordset contains 1000 basic words with hints to play Imposter with!
Made with Google Gemini because I didnt feel like making it myself, sue me

### Why does it come back after being deleted?

Because Godot (the game engine I built this app with) saves user data using a special folder you cant edit before exporting to mobile, I made it so it would take the default wordsets and add them on first launch, but I actually dont know if its first launch or not so it just happens every launch
