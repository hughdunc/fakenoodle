# Wordsets

There arent any wordsets here yet, but heres how do download them anyway!

## How to install files

### File Download Method
1. Download any JSON file(s) of your choice
2. Go into the app
3. Click "Words"
4. Click "Edit Word Sets"
5. Click "Import JSON File(s)"
6. Select the wordsets you want to install
7. Enable or disable them in the Words menu!

### Copy & Paste Method
1. Copy the contents of one of the JSON files
2. Go into the app
3. Click "Words"
4. Click "Edit Word Sets"
5. Click "Paste / Type JSON"
6. Paste the JSON you copied into the Text Box
7. Enable or disable them in the Words menu!

## How to export custom wordsets

1. Create a wordset, if not already
2. Go to the Words menu
3. Click "Edit Word Sets"
4. Click "Export Word Sets"
5. Select any wordsets you want to export as JSON
6. Send JSON files anywhere you want and have them use the Install guide here!

## How to submit wordsets to here

1. Export your wordset JSONs
2. Be logged into GitHub
3. Fork this repository
4. Go to "wordsets" branch
5. Three dots -> Upload File or Add File -> Upload Files
6. Upload your wordset JSONS
7. It will say something like "This branch is X Commit(s) ahead of `hughdunc/fakenoodle`"
8. Under that text, click "Contribute"
9. Click "Open Pull Request"
10. Add any notes and press "Create pull request"
11. I will review it and see if I want it on here, and if so, your wordset is available for anyone to use!
